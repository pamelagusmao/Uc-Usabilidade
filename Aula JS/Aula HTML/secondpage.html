Uc desenvolvimento de Jogos

A grade curricular do curso de Desenvolvimento de Jogos é voltada para capacitar os alunos a projetar, criar e programar jogos digitais para diferentes plataformas. O curso aborda disciplinas como lógica de programação, algoritmos, matemática aplicada, física para jogos e desenvolvimento gráfico em 2D e 3D. Além disso, os estudantes aprendem sobre design de jogos, modelagem de personagens, inteligência artificial para games e motores gráficos como Unity e Unreal Engine. A grade também inclui tópicos de storytelling, criação de mundos virtuais, e sonorização, preparando os alunos para criar experiências interativas e imersivas. Ao longo do curso, os estudantes participam de projetos práticos e colaborativos que simulam a criação de jogos reais, passando por todas as etapas do desenvolvimento.

Uc Analise de dados

O curso de Análise de Dados tem como objetivo capacitar profissionais para coletar, processar e interpretar grandes volumes de dados para tomar decisões estratégicas. A grade curricular inclui disciplinas como estatística, programação em Python e R, bancos de dados, machine learning, e visualização de dados. Os estudantes aprendem a manipular dados, construir modelos preditivos e utilizar ferramentas como SQL, Excel e Power BI para análise e apresentação de resultados. O curso também aborda conceitos de big data, análise de tendências e criação de dashboards interativos. A formação é altamente prática, com foco na aplicação de técnicas de análise em cenários reais, preparando os alunos para atuar em diversas áreas, como negócios, tecnologia e marketing.

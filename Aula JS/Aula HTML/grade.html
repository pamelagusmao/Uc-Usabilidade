Engenharia de Software

O curso de Engenharia de Software tem como objetivo formar profissionais capacitados para projetar, desenvolver, testar e manter sistemas de software complexos, eficientes e seguros. Durante o curso, os alunos aprendem as melhores práticas de desenvolvimento de software, desde o planejamento até a implementação, utilizando uma abordagem sistemática e organizada para resolver problemas de tecnologia.
